# E2E-ML-Usecases
End-to-End machine learning is concerned with preparing your data, training a model on it, and then deploying that model.

1) Pre-requisites and Resources.
2) Data Collection and Problem Statement.
3) Exploratory Data Analysis with Pandas and NumPy.
4) Selecting and Training Machine Learning Models.
5) Cross-Validation and Hyperparameter Tuning using Sklearn.
6) Saving the Trained Model.
7) Developing a web service.
